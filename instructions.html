How to Create and Print Documents

Welcome to the Multi-Links Associates Office Document System. This guide will help you create professional receipts, invoices, and letterheads, and export them as PDF files.

Getting Started

  1. Access the System: Navigate to the Documents page from the main menu
  2. Login: Enter your credentials when prompted to access the document templates
  3. Select Document Type: Choose from Payment Receipt, Invoice, or Letterhead tabs

Creating a Payment Receipt

  1. Click on the "Payment Receipt" tab in the document selector
  2. Fill in the receipt details:
    • Receipt No: Enter a unique receipt number (e.g., RCP-001)
    • Date: Select the payment date
    • Customer Name: Enter the customer's full name
    • Customer Address: Add the customer's complete address
    • Amount Received: Input the payment amount
    • Payment Method: Choose from Cash, Cheque, or Bank Transfer
  3. Review all information for accuracy
  4. Click the "Print" button to generate the PDF

Creating an Invoice

  1. Click on the "Invoice" tab in the document selector
  2. Complete the invoice information:
    • Invoice Number: Enter a unique invoice ID (e.g., INV-001)
    • Invoice Date: Set the issue date
    • Due Date: Specify the payment deadline
    • Client Information: Fill in client name and billing address
    • Service/Product Details: Add line items with descriptions, quantities, and rates
    • Tax Information: Include applicable taxes if required
  3. The system will automatically calculate subtotals and final amounts
  4. Review the invoice for completeness and accuracy
  5. Use the "Print" button to create the PDF

Creating a Letterhead

  1. Click on the "Letterhead" tab in the document selector
  2. Enter the letter information:
    • Date: Set the letter date
    • Reference: Add a reference number if needed
    • Recipient: Enter the recipient's name and address
    • Subject: Specify the letter subject
    • Content: Write your letter in the text area (includes professional signature template)
  3. Review the content for proper formatting and completeness
  4. Click "Print" to generate the PDF

Printing to PDF

Method 1: Browser Print Function

  1. Click the "Print" button on any document template
  2. In the print dialog, select "Save as PDF" or "Microsoft Print to PDF"
  3. Choose your save location and filename
  4. Click "Save"

Method 2: Keyboard Shortcut

  1. After clicking print, use Ctrl+P (Windows) or Cmd+P (Mac)
  2. Select "Save as PDF" from the destination dropdown
  3. Adjust print settings if needed:
    • Layout: Portrait (recommended)
    • Margins: Minimum or None for full-page documents
    • Scale: 100% for accurate sizing
  4. Save the PDF to your desired location
Important: Always preview your document before printing to ensure all information is correctly formatted and complete.

Tips for Best Results

Document Preparation

  • Complete All Fields: Fill in all required information before printing
  • Use Clear Button: Click "Clear" to reset forms when starting new documents
  • Check Preview: Review the document layout before printing
  • Consistent Numbering: Use sequential numbering for receipts and invoices
